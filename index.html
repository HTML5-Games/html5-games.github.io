About Us

We are the HTML5 Game Development Group, a group on Codecademy of over 50000 members, that is passionate about creating HTML5 games and moving the web forward. This site is a project by some of our members to create a website where anyone can create, share, learn how to make, and talk about HTML5 Games. This site is currently a work in progress.
